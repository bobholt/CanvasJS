Canvas JS

There is an 800x600 canvas here. Try messing around in firebug or other browser console.

ctx = new Canvas('ctx'); // creates a new Canvas object in the canvas with id="ctx"
ctx.circle({x:25, y:25, radius: 10, fillStyle:'rgb(200,0,0)'}); // creates a red circle
ctx.rectangle({x:50, y:75, height: 100, width: 50, fillStyle: 'rgba(0, 0, 200, 0.5)'}); // creates a bluish rectangle with transparency
ctx.getCurrentPos(); // returns the current x and y coordinates of the 'cursor'
