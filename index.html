Canvas JS

CanvasJS is a small helper library for the canvas 2d API. The goal is to extend and enhance the basic API while still remaining familiar.

Project home: http://roblarsen.github.com/CanvasJS/

Github: https://github.com/roblarsen/CanvasJS

There is an 800x600 canvas here. You can play with CanvasJS in browser console.

Here's an example of what it might look like. 

 var ctx = new Canvas("ctx");
    for ( var i = 0; i < 100; i++ ){
      var x = Math.random() * ctx.width,
          y = Math.random() * ctx.height,
          radius = Math.random() * 100;
      ctx.fillCircle({
        "x" : x,
        "y" : y,
        "radius" : radius,
        "fillStyle" : '#'+Math.floor(Math.random()*16777215).toString(16)
      });
    }
